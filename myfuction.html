JavaScript Concepts Tutorial

1. Function

A function is a block of reusable code that can be called to perform a specific task.

function greet(name) {
    return "Hello, " + name + "!";
}

const greeting = greet("John");
console.log(greeting); // Output: "Hello, John!"

2. Function Parameters and Arguments

Parameters are variables defined in a function's declaration. Arguments are values passed into a function when it's called.

function add(a, b) {
    return a + b;
}

const result = add(5, 3);
console.log(result); // Output: 8

3. Default Parameter

Default parameters provide default values for function parameters if no argument is provided.

function greet(name = "Guest") {
    return "Hello, " + name + "!";
}

const greeting = greet();
console.log(greeting); // Output: "Hello, Guest"

4. Rest Parameter

The rest parameter allows you to pass an arbitrary number of arguments to a function as an array.

function sum(...numbers) {
    return numbers.reduce((total, num) => total + num, 0);
}

const result = sum(1, 2, 3, 4, 5);
console.log(result); // Output: 15

5. Spread Parameter

The spread parameter allows you to expand an array or object into individual elements or properties.

const numbers = [1, 2, 3];
const newNumbers = [...numbers, 4, 5];
console.log(newNumbers); // Output: [1, 2, 3, 4, 5]

6. Arrow Function

Arrow functions provide a concise syntax for defining functions.

const multiply = (a, b) => a * b;
console.log(multiply(2, 3)); // Output: 6

7. Nested Function

Nested functions are functions defined within other functions.

function outer() {
    function inner() {
        console.log("Inside inner function.");
    }
    
    console.log("Inside outer function.");
    inner();
}

outer();

8. Function Scope

Function scope refers to the visibility and accessibility of variables within a function.

function exampleScope() {
    const x = 10;
    
    if (x > 5) {
        const y = 20;
        console.log(x); // Output: 10
        console.log(y); // Output: 20
    }
    
    console.log(x); // Output: 10
    // console.log(y); // Error: y is not defined
}

exampleScope();

9. Closures

Closures are functions that have access to variables from their outer (enclosing) function's scope.

function outer() {
    const message = "Hello, ";

    function inner(name) {
        return message + name;
    }

    return inner;
}

const greetJohn = outer();
console.log(greetJohn("John")); // Output: "Hello, John"

10. Callback Function

A callback function is a function passed as an argument to another function and executed later.

function greet(name, callback) {
    const message = "Hello, " + name + "!";
    callback(message);
}

function displayMessage(message) {
    console.log(message);
}

greet("Sarah", displayMessage); // Output: "Hello, Sarah!"

11. Higher-Order Function

Higher-order functions are functions that take other functions as arguments or return them.

function multiplyBy(factor) {
    return function (number) {
        return number * factor;
    };
}

const double = multiplyBy(2);
console.log(double(5)); // Output: 10

12. Pure Function

A pure function always produces the same output for the same input and doesn't have side effects.

function add(a, b) {
    return a + b;
}

console.log(add(2, 3)); // Output: 5
console.log(add(2, 3)); // Output: 5 (Same input, same output)

13. IIFE (Immediately Invoked Function Expression)

An IIFE is a self-executing function that runs immediately after being defined.

(function () {
    const message = "I am an IIFE!";
    console.log(message);
})(); // Output: "I am an IIFE!"

14. Call Stack

The call stack is a data structure that keeps track of function calls in a program.

function first() {
    console.log("First function");
    second();
}

function second() {
    console.log("Second function");
}

first();

15. Recursion Function

Recursion is a programming technique where a function calls itself to solve a problem.

function factorial(n) {
    if (n === 0) {
        return 1;
    }
    return n * factorial(n - 1);
}

console.log(factorial(5)); // Output: 120
